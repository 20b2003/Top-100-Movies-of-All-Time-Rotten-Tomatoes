This task consist of extracting data from a website, restructure the format, clean the HTML tags and convert it into a csv. file. This can be known as web scrapping.

The website to do web scrapping for this task is: https://www.rottentomatoes.com/top/bestofrt/

What I do is write down the code that search the contents from a table in a webpage, and storing that contents by appending in a csv file. Its also remove the \n character while copying contents. It also search for tr and td tags, that means searching an HTML document.
